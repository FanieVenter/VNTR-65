# **VNTR-65**
A custom, over-engineered 65% keyboard featuring an RP2350 MCU, a hall effect key matrix, an analog slide potentiometer, an OLED display. Built for daily use thanks to HackClub.

## Why I decided to make this keyboard.
I wanted to build a nice keyboard that I could use every day at school while knowing it won't break, I also wanted to make a keyboard that used hall effect switches to allow for analog inputs in racing ganes like Assetto Corsa and giving me a more challenging and fun experience designing it.

## How to use it.
To use this keyboard you will need to either download the production files for JLCPCB or create your own by exporting it them from Kicad. You will then need to place a order for the PCB using the productions files at your chosen pcb manufacturer. After you have received the PCB you will need to solder all the components onto the PCB and then flash the firmware onto the RP2350. While this is happening you can either 3d print the case or get it milled out of aluminum and wood. After you have the case and the PCB ready you can then assemble the keyboard and upload the firmware to the RP2350 and then you can use it.
